How it works

Four layers.
One reliable experience.

From the moment a homeowner describes a problem to the moment it's resolved — every step is powered by structured intelligence.

01
🧠

Describe the problem

Homeowners describe the issue in plain language. Claude AI parses the intake — extracting trade, severity, urgency, and risk flags — with a zero-dependency fallback if the LLM is unavailable.

02
⚡

Smart routing fires

The routing engine applies urgency rules, capacity filters, suspension exclusions, and preference memory to rank the best available contractors in real time.

03
🤖

Agents watch the job

ETA watchdogs, anomaly detectors, and price variance monitors run every 5 seconds. If something goes wrong, the agent proposes a re-match — one tap to accept.

Intelligence layer

Every feature built around outcome data.

Reliability Scoring
Live trust scores driven by real job outcomes — not reviews gameable by the contractor.
Smart Routing
Declarative routing rules that filter by urgency, capacity, suspensions, and homeowner history.
Proactive Notifications
ETA alerts, re-match proposals, anomaly flags — typed and color-coded by category.
Preference Memory
The platform learns which contractors a homeowner trusts and boosts them in future matches.

Autonomy layer

An agent that never sleeps.

While a job is in progress, TrustFix's autonomous agent is watching. It detects delays, flags anomalies, proposes re-matches, and suspends underperforming contractors — all without requiring homeowner input.
